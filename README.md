# Pengubah Nama XLXS

## Note:
- Program ini cuma untuk ubah nama, tambah row baru dengan style dari row sebelumnya,
  dan hapus row tidak berguna
- Nilai dari cell excel dihapus mandiri
- Spacing di akhir perlu diatur sendiri
- Kontak: email: argya.menuntut.ilmu@gmail.com atau ig: @argya.rayyan

## Cara guna:
- Download ***Pengubah.Nama.XLSX.exe*** di Release 
- Pindahkan ***Pengubah.Nama.XLSX.exe*** ke folder/directory-nya sendiri (optional)
- Siapkan file XLSX asli untuk diedit (input)
- Siapkan file nama baru (untuk format cek sub-judul selanjutnya)
- Jalankan ***Pengubah.Nama.XLSX.exe*** dan ikuti perintahnya

## Format "Nama baru txt"
  Nama baru ditaruh didalam file txt dengan format: <br /> <br />
  ```
  *NAMA SPREADSHEET 1*:
  *NAMA 1*
  *NAMA 2*
  ...
          (Enter/Newline)
  *NAMA SPREADSHEET 2*:
  *NAMA 1*
  ...
  ```
  Contoh:
  ```
  BPH:    #Nama spreadsheet yang pertama + ':'
  H M A   #Nama 1 (inisial) (di aslinya jangan inisial)
  S A K   #Nama 2 (inisial)
              #Enter/New line
  PMS:    #Nama spreadsheet kedua + ':'
  A R M   #Nama 1 (inisial)
  M R C   #Nama 1 (inisial)
  G F A W #Nama 1 (inisial)

  ```
